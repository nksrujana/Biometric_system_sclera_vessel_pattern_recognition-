Installation and Setup Guide:

1\. Prerequisites: Python installed on a Windows Environment or use Google Colab. A camera for image acquisition.

2\. Install Required Libraries:/pip install torch opencv-python numpy matplotlib

3\. Running the System:

* Preprocessing: Run the script to normalize and remove noise from your eye images.
* Segmentation: Execute the U-Net model to generate sclera masks.
* Authentication: Run the main application to match live captures against your database templates.
